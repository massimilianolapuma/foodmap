# 0005 — P11: Export recipes and meal plan to PDF

- **Date:** 2026-05-31
- **Issue:** #49
- **Branch:** `feat/49-pdf-export`

## Goal

Let the user export the current meal plan and its full recipes to a shareable,
localized PDF (ingredients, kcal, steps, time).

## Changes

- `FoodMap/Features/MealPlanner/MealPlanPDFRenderer.swift` (new)
  - `MealPlanPDFRenderer.render(_:)` paginates an A4 PDF with
    `UIGraphicsPDFRenderer`, drawing the plan title/subtitle and, per meal,
    name, meta (day · type · min · kcal), summary, ingredients and numbered
    steps. A private `Layout` cursor handles word-wrap and page breaks.
  - `MealPlanPDF: Transferable` wraps the PDF bytes for `ShareLink` export
    (surfacing `UIActivityViewController`), with a suggested filename.
- `FoodMap/Features/MealPlanner/MealPlannerView.swift`
  - Added a toolbar `ShareLink` (`meals.exportPDFButton`) that exports the
    rendered `MealPlanPDF`, plus `planSignature`/`refreshPDF()` to regenerate
    the document whenever the plan or its meals change (generation or swap).
- Localized strings (en/it): `Export PDF`, `Meal plan`, `Starting %@`.
- `FoodMapTests/MealPlanPDFRendererTests.swift` (new): valid `%PDF-` header,
  empty-plan still renders, and large plans paginate (grow in size).
