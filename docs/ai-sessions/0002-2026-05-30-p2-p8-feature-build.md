## Decisions / notes
- FoundationModels code path is real (`#if canImport(FoundationModels)` + `if #available(iOS 26, *)`)
  but the simulator falls back to the rule-based planner, so tests stay deterministic.
- Localization uses classic `.lproj/Localizable.strings` (not `.xcstrings`) because
  XcodeGen reliably builds variant groups + adds `it` to `knownRegions` that way.
- Privacy manifest declares no tracking, no data collected, no required-reason APIs
  (none used). Sensitive diet/allergy data stays on-device.
- Each phase committed separately on `main` with `Closes #n`, pushed green.
