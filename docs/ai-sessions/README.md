# AI Session Logs

Chronological record of what each AI working session accomplished, so context
is never lost between conversations. **Newest file reflects the latest state.**

## Convention
- One file per session: `NNNN-YYYY-MM-DD-short-slug.md` (zero-padded sequence).
- Start from the template in `_TEMPLATE.md`.
- Keep it short: what changed, why, what's verified, what's next.
- Link issues/PRs/commits where relevant.
